## Installation

Install the dependencies:

```bash
npm install
```

Set the environment variables:

```bash
cp .env.example .env

# open .env and modify the environment variables (if needed)
```

## Commands

Running locally:

```bash
npm run dev
```

Running in production:

```bash
npm run start:dev
```

Testing:

```bash
# run all tests
npm run test
```

Build:

```bash
npm run build
```
